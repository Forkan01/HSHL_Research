# 1. Clone the Kuksa.val Repository:
Open Terminal:
- git clone https://github.com/eclipse/kuksa.val.git

# 2. Build and Run the Kuksa.valServer:
Open Docker & minimize.
Go back to the Terminal:
- cd kuksa.val
- docker-compose up --build

# 3. Keep the manual_control.py & speed_provider.py in a folder & open in VS code.
create an environment and install:
- Carla, kuksa-client, pygame, numpy

# 4. Open Carla, Run manual_control.py & Run speed_provider.py in another terminal.
